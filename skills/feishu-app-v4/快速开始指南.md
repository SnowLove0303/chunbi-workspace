# 飞书应用创建自动化 - 快速开始指南

> ⏱️ 5 分钟完成环境配置，2 分钟创建飞书应用！

## 第一步：安装环境（3 分钟）

### 1. 检查 Python
打开 CMD，输入:
```cmd
python --version
```
如果显示版本号（如 `Python 3.10.0`），说明已安装。否则请访问 [python.org](https://www.python.org/downloads/) 下载安装。

### 2. 运行安装脚本
进入技能包目录，双击运行:
```
install.bat
```
等待安装完成（约 2-3 分钟）。

## 第二步：启动 Chrome 调试模式（1 分钟）

### 方法 A: 使用快捷方式（推荐）
1. 右键桌面 → 新建 → 快捷方式
2. 粘贴以下路径:
```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\Users\%USERNAME%\AppData\Local\Google\Chrome\User Data"
```
3. 命名快捷方式为"Chrome 调试模式"
4. 双击运行该快捷方式

### 方法 B: 命令行启动
打开 CMD，复制粘贴:
```cmd
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\Users\%USERNAME%\AppData\Local\Google\Chrome\User Data"
```

## 第三步：登录飞书（1 分钟）

在打开的 Chrome 浏览器中:
1. 访问 https://open.feishu.cn/app
2. 使用飞书 App 扫码登录
3. **保持浏览器窗口打开，不要关闭**

## 第四步：运行自动化（2 分钟）

回到技能包目录，双击运行:
```
run.bat
```

脚本会自动执行以下步骤:
- ✅ 检查登录状态
- ✅ 创建应用（名称："飞书 Flow 测试"）
- ✅ 添加机器人功能
- ✅ 导入 196 个权限
- ✅ 配置事件订阅
- ✅ 获取 APP ID

## 第五步：保存凭证（重要！）

当脚本提示"请点击 APP Secret 旁边的眼睛图标"时:
1. 在浏览器中找到 APP Secret
2. 点击眼睛图标查看
3. 点击复制按钮
4. **立即保存到安全位置**（密码管理器或加密文档）

⚠️ **APP Secret 只显示一次，丢失后无法找回！**

## 第六步：发布应用

脚本会暂停并等待人工确认:
1. 检查应用配置是否正确
2. 在浏览器中点击"发布版本"按钮
3. 回到命令行，按回车键继续

## 完成！🎉

查看结果:
- 📸 截图文件：`screenshots\` 目录
- 📝 执行日志：`logs\` 目录
- 📊 执行报告：`logs\report_*.txt`

## 常见问题

### Q: Chrome 无法启动调试模式？
A: 确保没有其他 Chrome 窗口在运行，先全部关闭再重新启动。

### Q: 脚本报错"连接失败"？
A: 检查 Chrome 是否正常运行，以及是否在地址栏看到"您使用的是不受支持的命令行标记"提示。

### Q: 权限导入不完整？
A: 部分权限需要管理员审批，这是正常现象。查看日志确认哪些权限待审批。

### Q: 如何修改应用名称？
A: 编辑 `feishu_cdp_automation.py`，找到 `CONFIG["app_name"]` 修改即可。
